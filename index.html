Qosımtalar dúnyasına xosh keldińiz!

Qaraqalpaq tilindegi qosımtalardı úyreniw ańsat hám qızıqlı. Bul sabaqta biz sóz jasawshı hám sóz túrlewshi qosımtalardı kórip shıǵamız.

Túbir + Qosımta = Jańa máni

Sóz jasawshı

Sózge jańa máni berip, jańa sóz jasaydı. Mısalı: Balıq -> Balıqshı.

Kópshilik

Zattıń birewden kóp ekenligin bildiredi. Mısalı: Kitap -> Kitaplar.

Septikler

Sózlerdi bir-biri menen baylanıstıradı. Mısalı: Mektep -> Mektepke.

Qosımtalar túrleri

Sóz jasawshı qosımtalar

Bul qosımtalar túbir sózge jalǵanıp, onnan pútkilley jańa mánige iye bolǵan basqa sóz jasaydı.

-shı / -shi

Kásip-óner atların jasaydı.

  • Etik + shi = Etikshi
  • Qoy + shı = Qoyshı
-lıq / -lik

Zatlıq at yaki sapalıq máni bildiredi.

  • Dos + lıq = Doslıq
  • Tegis + lik = Tegislik
-sız / -siz

Joqlıq mánisin bildiredi.

  • Suw + sız = Suwsız
  • Bilim + siz = Bilimsiz

Sóz túrlewshi qosımtalar

Sózdiń leksikalıq mánisin ózgertpeydi, tek onıń grammatikalıq formasın ózgertedi.

Kópshilik jalǵawı (-lar / -ler)

Juwın dawıslılarǵa -lar, jińishke dawıslılarǵa -ler jalǵanadı.

Qatatań:

Alma -> Almalar

Oqıwshı -> Oqıwshılar

Jińishke:

Keme -> Kemeler

Múǵallim -> Múǵallimler

Táweldik jalǵawları (Meniń, seniń...)

Bet Birlik Mısal (Qol)
I bet (Men) -ım / -im Qolım
II bet (Sen) -ıń / -iń Qolıń
III bet (Ol) -ı / -i Qolı

Septik jalǵawları

Qaraqalpaq tilinde 6 septik bar. Olar sózlerdi baylanıstıradı.

1. Ataw (Kim? Ne?)

Jalǵawı joq.

Kitap stolda tur.

2. İlik (Kimniń? Neniń?)

-nıń, -niń, -dıń, -tiń...

Mekteptiń hawlısı.

3. Barıs (Kimge? Nege?)

-ǵa, -ge, -qa, -ke, -na, -ne

Qalaǵa baraman.

4. Tabıs (Kimdi? Neni?)

-nı, -ni, -dı, -ti...

Nandı al.

5. Shıǵıs (Kimnen? Neden?)

-nan, -nen, -dan, -ten...

Jumıstan qayttım.

6. Orın (Kimde? Nede?)

-da, -de, -ta, -te, -nda

Úyde otırman.

Bilimińizdi sınań!

Durıs juwaptı tańlań.
